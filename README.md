# exchange_rates
____
**Get started**

Clone repository
Install requirements:

    `pip install -r requirements.txt`

Migrate command:

    `python manage.py migrate`

Start a redis via docker:

    `docker run -p 6379:6379 -it redis/redis-stack:latest` 

Runserver command:

    `python manage.py runserver`
